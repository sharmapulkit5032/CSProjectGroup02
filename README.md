# CSProjectGroup02 - Patient Clinical Details Project

This is a C# project for the Mini-Final-Project of COMP2131.

## Project Files
- **Program.cs**: Main program that creates patient objects and displays their details.
- **Patient.cs**: Class file containing patient properties (First Name, Last Name, Weight, Height), BMI calculation method, and blood pressure check method.
- **CSProjectGroup02.sln**: Visual Studio solution file.

## Notes
- Run Program.cs to see the console output.
- All files are properly structured for submission.
